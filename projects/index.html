Projects
Hypixel News
Hypixel News is a work-in-progress bot that sends you the latest news from the Hypixel Network. This project is not affiliated nor associated with Hypixel Inc.
Iris
Iris is a browser extension for Hypixel made to check any Hypixel player's last session easily and quickly. Compatible with Chromium based browsers and Firefox.
NotifHy
NotifHy is notification bot for Hypixel. Share logins with friends, get reminders to claim daily rewards, and get alerts on suspicious activity.
Shorts Deflector
Shorts Deflector is a Chromium and Firefox browser extension that redirects YouTube Shorts to the normal desktop interface.
